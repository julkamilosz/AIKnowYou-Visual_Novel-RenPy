# AIKnowYou-Visual_Novel-RenPy
Projekt akademicki, realizowany samodzielnie (Semestr III)
Rola: Game Designer & Narrative Programmer
Zakres prac:
- Zaprojektowanie i zaimplementowanie nieliniowej struktury narracyjnej z rozgałęzieniami fabularnymi i wieloma zakończeniami.
- Programowanie logiki gry w języku skryptowym silnika Ren'Py (zarządzanie stanem gry, obsługa zmiennych wejściowych gracza, integracja assetów audio-wizualnych).
- Stworzenie mechaniki dynamicznego budowania napięcia (skrypty sterujące muzyką, efektami dźwiękowymi oraz przejściami ekranowymi typu fade/flash).
